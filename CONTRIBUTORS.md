## How to Contribute

We welcome contributions from everyone! Here's how you can help:

### Ways to Contribute

- 🐛 **Report bugs** - Found a bug? [Open an issue](https://github.com/hazel-js/hazeljs/issues/new?template=bug_report.yml)
- ✨ **Suggest features** - Have an idea? [Request a feature](https://github.com/hazel-js/hazeljs/issues/new?template=feature_request.yml)
- 📝 **Improve docs** - Help us make the documentation better
- 💻 **Submit PRs** - Fix bugs or add features
- 🎨 **Share examples** - Built something cool? Show it off!
- ⭐ **Star the repo** - Show your support
- 💬 **Join discussions** - Help others in [Discussions](https://github.com/hazel-js/hazeljs/discussions)

### Recognition
