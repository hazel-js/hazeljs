# Contributing to Awesome HazelJS

Thank you for your interest in contributing to Awesome HazelJS!

## Guidelines

### Adding to the List

- **Search previous suggestions** before making a new one to avoid duplicates
- **Make an individual pull request** for each suggestion
- Use the following format: `[Name](link) - Description`
- **Keep descriptions short and simple**, but descriptive
- Start the description with a capital and end with a period
- Check your spelling and grammar
- New categories or improvements to the existing categorization are welcome

### Quality Standards

To be added to this list, projects should:

- Be related to HazelJS
- Be actively maintained (or clearly marked as archived/unmaintained)
- Have clear documentation
- Follow best practices
- Be useful to the community

### Updating Your PR

If the maintainers notice anything that needs to be changed, they will ask you to edit your PR. There's no need to open a new PR, just edit the existing one.

## How to Submit

1. Fork this repository
2. Add your item to the appropriate category in README.md
3. Ensure your addition follows the format and guidelines
4. Create a pull request with a clear title and description
5. Wait for review

## Categories

When adding items, please place them in the most appropriate category:

- **Official Resources** - Only official HazelJS resources
- **Community Projects** - Projects built with HazelJS
- **Tutorials & Guides** - Educational content
- **Packages & Plugins** - Extensions and integrations
- **Tools** - Development and deployment tools
- **Examples** - Code examples and demos
- **Videos** - Video content
- **Articles** - Written content
- **Community** - Community resources

## Questions?

If you have any questions, feel free to:
- Open an issue
- Ask in [GitHub Discussions](https://github.com/hazel-js/hazeljs/discussions)
- Join our [Discord](https://discord.gg/rnxaDcXx)

Thank you for contributing! 🎉
